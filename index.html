Política de Privacidad — LoL Clean

Última actualización: 22 de marzo de 2025

1. Información que recopilamos

LoL Clean solicita al usuario su nombre de invocador (Riot ID) y región de juego. Esta información se utiliza exclusivamente para consultar la API pública de Riot Games y calcular los días transcurridos desde la última partida.

2. Cómo usamos la información

Los datos introducidos se utilizan para:

  • Consultar la API de Riot Games y obtener la fecha de la última partida.
  • Mostrar el contador de días sin jugar dentro de la aplicación.
  • Almacenar el resultado en nuestra base de datos (Supabase) para futuras consultas.

No vendemos, compartimos ni cedemos ningún dato a terceros con fines comerciales.

3. Servicios de terceros

La aplicación hace uso de los siguientes servicios externos:

  • Riot Games API — para obtener el historial de partidas. Consulta su política de privacidad en riotgames.com.
  • Supabase — como base de datos en la nube. Consulta su política en supabase.com.

4. Almacenamiento de datos

El Riot ID y la región del usuario se almacenan en los servidores de Supabase para evitar que el usuario tenga que introducirlos en cada sesión. No se almacena ninguna contraseña ni credencial de cuenta de Riot Games.

5. Derechos del usuario

Puedes solicitar la eliminación de tus datos en cualquier momento contactándonos por correo electrónico.

7. Cambios en esta política
