Privacy Policy — LoL Clean

Last updated: March 22, 2025

1. Information We Collect

LoL Clean asks the user for their Riot ID (game name) and region. This information is used solely to query the public Riot Games API and calculate the number of days since the last match.

2. How We Use the Information

The data entered is used to:

  • Query the Riot Games API to retrieve the date of the last match played.
  • Display the days-without-playing counter inside the app.
  • Store the result in our database (Supabase) for future queries.

We do not sell, share, or transfer any data to third parties for commercial purposes.

3. Third-Party Services

The app uses the following external services:

  • Riot Games API — to retrieve match history. See their privacy policy at riotgames.com.
  • Supabase — as a cloud database. See their policy at supabase.com.

4. Data Storage

The user's Riot ID and region are stored on Supabase servers to avoid re-entering them on every session. No Riot Games account passwords or credentials are stored.

5. User Rights

You may request deletion of your data at any time by contacting us via email.
